== Description ==


### Simple WordPress Sponser Link Plugin

Sponsered Link is the best WordPress plugin. Sponsered Link allow you to easily create and manage Sponser Link through a simple admin interface.

You can dislay random sponser link in wordPress post, page or widget.


== SponserLink plugin features include ==

- **NEW** Display SponserLink as default option is on settings page
- pagination setting 
- display sponser link on front page 

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload `sponseredLink` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Create new SponserLink in the `Sponser Link` Panel.
1. Place `<?php do_shortcode('[SponseredLink]'); ?>` in your templates
1. use shortcode [SponseredLink] in page,post, widget
